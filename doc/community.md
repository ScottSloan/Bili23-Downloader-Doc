## ⚠️ 提问的智慧 (必读)

::: warning 为了大家能快速定位并解决你的问题，提问前请务必确认：
请**避免只发一句“为什么解析不了了”、“我这报错了怎么办”**，通常这无法让群友或作者得知出错根源。

一次高效的求助，请务必在一开始就提供以下关键信息：
1. **具体现象**：发生了什么错误？是在点击“解析”时报错，还是在“开始下载”后报错？
2. **测试链接**：如果是特定甚至少数的视频无法下载，请顺手**附上出错视频的 B 站原链接**以便我们复现。
3. **系统与版本**：说明你的操作系统（如 Windows 11 / macOS）以及当前使用的软件版本号。
4. **日志或清晰截图**：请务必提供**完整的系统报错日志**（优先级最高），或者截取包含完整错误提示信息的界面截图（请勿使用手机拍摄屏幕）。
:::
